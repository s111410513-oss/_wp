# 作業 05 - 網誌系統 (Blog)

## 說明

這是一個使用 Node.js + Express 開發的 Threads 風格網誌系統。

## 檔案結構

- `blog/` - 基礎網誌
- `blog 1/` - 網誌版本 1
- `blog 2/` - 網誌版本 2  
- `blog 3/` - 網誌版本 3 (最新)

## 功能特色

- Threads 風格 UI 設計
- 個人貼文區與公共貼文區
- 按讚、收藏、分享功能
- 點擊作者查看個人資料
- Hashtag 支援
- 響應式設計

## 運行方式

```bash
cd blog 3
npm install
npm start
```

然後在瀏覽器開啟 http://localhost:3001

## 技術棧

- Node.js
- Express.js
- Vanilla JavaScript
- CSS3
